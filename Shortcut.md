# Visual Studio快捷键
```txt
CTRL+K,D  格式化当前文档
CTRL+K,C  注释
CTRL+K,U  取消注释
CTRL+M,M  切换大纲显示

F9        设置/取消断点
F5        开始调试
SHIFT+F5  取消调试
F10       逐过程单步调试
F11       逐语句单步调试
SHIFT+F11 跳出
F12       转到定义

CTRL+F        当前文件查找
CTRL+H        当前文件替换
CTRL+SHIFT+F  在文件中查找
CTRL+SHIFT+H  在文件中替换
```

# Android Studio快捷键
```txt
CTRL+Q        显示注释文档
CTRL+ALT+L    格式化
CTRL+/        行注释/取消注释
CTRL+SHIFT+/  块注释/取消注释
CTRL+B        转到定义
CTRL+ALT+left/right 返回至上次浏览的位置
ALT+left/right      TAB页切换
CTRL+H        显示类结构

CTRL+F        在当前文件查找
CTRL+R        在当前文件替换
CTRL+SHIFT+F  在文件中查找
CTRL+SHIFT+R  在文件中替换

SHIFT+F9      开始调试
CTRL+F2       停止调试
F8       逐过程单步调试
F7       逐语句单步调试
SHIFT+F8 跳出
```
