# Visual Studio快捷键
```txt
CTRL+K,D  格式化当前文档
CTRL+K,C  注释
CTRL+K,U  取消注释
CTRL+M,M  切换大纲显示

F9        设置/取消断点
F5        开始调试
SHIFT+F5  取消调试
F10       逐过程单步调试
F11       逐语句单步调试
SHIFT+F11 跳出
F12       转到定义

CTRL+F        当前文件查找
CTRL+H        当前文件替换
CTRL+SHIFT+F  在文件中查找
CTRL+SHIFT+H  在文件中替换
```

# Android Studio快捷键
```txt
CTRL+Q        显示注释文档
CTRL+ALT+L    格式化
CTRL+/        行注释/取消注释
CTRL+SHIFT+/  块注释/取消注释
CTRL+B        转到定义
CTRL+ALT+left/right 返回至上次浏览的位置
ALT+left/right      TAB页切换
CTRL+H        显示类结构
CTRL+F12      显示当前文档的结构
CTRL+P        显示参数信息
CTRL+W        选择单词->语句->函数
CTRL+N        快速转到当前文件中的类定义

CTRL+F        在当前文件查找
CTRL+R        在当前文件替换
CTRL+SHIFT+F  在文件中查找
CTRL+SHIFT+R  在文件中替换

SHIFT+F9      开始调试
F9            程序断点停住后，继续执行
CTRL+F2       停止调试
F8       逐过程单步调试
F7       逐语句单步调试
SHIFT+F8 跳出
```

# Ollydbg快捷键
```txt
F3       打开文件进行调试
F2       添加/移除断点
Ctrl+F2  重新运行正在被调试的程序
Alt+F2   关闭正在被调试的程序

F7   单步步入
F8   单步步过
F9   让程序运行到下一个断点

Alt+B   显示断点窗口
Alt+C   显示CPU窗口
Alt+E   显示模块列表
Alt+K   显示调用栈窗口
Alt+L   显示日志窗口
Alt+M   显示内存窗口

Alt+F9  回到用户程序的领空

Ctrl+B  二进制搜索
Ctrl+F  汇编指令搜索
Ctrl+G  跟随表达式（可以是win32 API）

;       在汇编指令后写入注释
空格     修改该条汇编指令
```
